🗄️ Supabase Database Configuration

Step 1 — Create Free Supabase Account
Go to supabase.com → New Project → Choose a name & password → Wait ~2 min for setup
Step 2 — Run this SQL in Supabase SQL Editor
Go to SQL Editor in your Supabase dashboard and run:
-- Users table CREATE TABLE hr_users ( id BIGSERIAL PRIMARY KEY, username TEXT UNIQUE NOT NULL, password TEXT NOT NULL, full_name TEXT, role TEXT DEFAULT 'user', status TEXT DEFAULT 'active', created_at DATE DEFAULT CURRENT_DATE ); -- Insert default admin INSERT INTO hr_users (username,password,full_name,role,status) VALUES ('admin','admin123','Super Admin','admin','active'); -- Results table CREATE TABLE hr_results ( id BIGSERIAL PRIMARY KEY, session_id TEXT, job_title TEXT, candidate_name TEXT, score INTEGER, ats_rating TEXT, result_json JSONB, created_by TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Allow public read/write (Row Level Security off for simplicity) ALTER TABLE hr_users DISABLE ROW LEVEL SECURITY; ALTER TABLE hr_results DISABLE ROW LEVEL SECURITY;
Step 3 — Get Your Credentials
Go to Project Settings → API and copy:
  • Project URL (looks like https://xxxx.supabase.co)
  • anon/public key (long JWT string)
